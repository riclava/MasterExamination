# 计算机网络满分知识点与练习题汇总

## 1. VLAN 与 IEEE 802.1Q

### 必须掌握

VLAN 是**逻辑局域网**，与物理位置无关。
IEEE 802.1Q 定义 VLAN 标准，会在以太网帧中加入 VLAN Tag。
VLAN 可以隔离广播域，减少广播风暴影响。

### 练习题

**1. 下列关于 VLAN 的说法，错误的是：**

A. VLAN 可以隔离广播域
B. VLAN 一定按照物理位置划分
C. IEEE 802.1Q 定义了 VLAN 标签
D. VLAN 帧格式与普通以太网帧格式不同

答案：B

**2. VLAN 的主要作用是什么？**

A. 增加 IP 地址数量
B. 提高 CPU 计算速度
C. 划分逻辑广播域
D. 替代 TCP 协议

答案：C

---

## 2. 以太网帧与 VLAN Tag

### 必须掌握

普通以太网帧没有 VLAN Tag。
802.1Q 帧会插入 4 字节 VLAN 标签。
VLAN ID 用于标识帧属于哪个 VLAN。

### 练习题

**1. IEEE 802.1Q 在以太网帧中增加了什么字段？**

A. IP 地址字段
B. VLAN 标签字段
C. TCP 端口字段
D. DNS 字段

答案：B

**2. VLAN Tag 的作用主要是：**

A. 标识帧所属的 VLAN
B. 标识 TCP 连接
C. 标识浏览器进程
D. 标识 HTTP 请求类型

答案：A

## 3. ARP、广播与单播

### 必须掌握

ARP 用于通过 IP 地址查询 MAC 地址。
ARP 请求通常是广播。
ARP 响应通常是单播。
注意：**ARP 不属于 UDP**，它工作在网络层和数据链路层之间。

### 练习题

**1. ARP 请求一般采用什么方式发送？**

A. 单播
B. 广播
C. 组播
D. 任播

答案：B

**2. ARP 协议的主要作用是：**

A. 根据域名查询 IP 地址
B. 根据 IP 地址查询 MAC 地址
C. 根据端口号查询进程
D. 根据 MAC 地址查询端口号

答案：B

**3. 下列说法正确的是：**

A. ARP 是 TCP 协议
B. ARP 是 UDP 协议
C. ARP 请求通常是广播
D. ARP 响应一定是广播

答案：C

## 4. 802.11、CSMA/CA、RTS/CTS

### 必须掌握

无线局域网 IEEE 802.11 的 MAC 层使用 **CSMA/CA**。
有线以太网常见的是 CSMA/CD。
无线网络难以检测碰撞，因此采用冲突避免。
RTS/CTS 可用于缓解隐藏终端问题。

### 练习题

**1. IEEE 802.11 无线局域网 MAC 层采用的协议是：**

A. CSMA/CD
B. CSMA/CA
C. PPP
D. HDLC

答案：B

**2. RTS 帧的含义是：**

A. Request To Send
B. Ready To Save
C. Route To Server
D. Receive To Send

答案：A

**3. 无线局域网中使用 RTS/CTS 机制主要是为了缓解：**

A. 域名解析问题
B. 隐藏终端问题
C. IP 地址耗尽问题
D. 端口冲突问题

答案：B

## 5. CIDR 地址块与子网掩码

### 必须掌握

CIDR 形式：

```text
a.b.c.d/n
```

其中 `/n` 表示前 n 位是网络位。
主机位数：

```text
32 - n
```

例如：

```text
/20
```

表示网络位 20 位，主机位 12 位，掩码是：

```text
255.255.240.0
```

### 练习题

**1. 地址块 192.168.0.0/20 中，主机位有多少位？**

答案：12 位

**2. /22 对应的子网掩码是：**

A. 255.255.248.0
B. 255.255.252.0
C. 255.255.254.0
D. 255.255.255.0

答案：B

**3. /19 对应的子网掩码是：**

A. 255.255.192.0
B. 255.255.224.0
C. 255.255.240.0
D. 255.255.248.0

答案：B

## 6. CIDR 路由聚合

### 必须掌握

路由聚合的核心是：
把多个网络地址转成二进制，找它们的**最长共同前缀**。

例如：

```text
10.18.96.0/21
10.18.104.0/21
10.18.112.0/21
10.18.120.0/21
```

它们可以聚合为：

```text
10.18.96.0/19
```

### 练习题

**1. 以下两个地址块：**

```text
192.168.0.0/24
192.168.1.0/24
```

可以聚合为：

答案：

```text
192.168.0.0/23
```

**2. 以下四个地址块：**

```text
10.0.0.0/24
10.0.1.0/24
10.0.2.0/24
10.0.3.0/24
```

可以聚合为：

答案：

```text
10.0.0.0/22
```

**3. 下面哪个地址块能覆盖 172.16.8.0/24 到 172.16.15.0/24？**

A. 172.16.8.0/21
B. 172.16.8.0/22
C. 172.16.0.0/24
D. 172.16.16.0/21

答案：A

## 7. 子网划分

### 必须掌握

子网划分要同时满足两个条件：

1. 子网数量够不够
2. 每个子网主机数量够不够

公式：

```text
子网数 = 2^子网位数
可用主机数 = 2^主机位数 - 2
```

例如 B 类地址默认 `/16`，如果借 6 位做子网位，则变成 `/22`。

```text
/22 = 255.255.252.0
```

主机位还有 10 位：

```text
2^10 - 2 = 1022
```

### 练习题

**1. 某 B 类网络要划分至少 50 个子网，每个子网至少 500 台主机，应至少借几位作为子网位？**

答案：6 位。因为 2⁵ = 32 不够，2⁶ = 64 够。

**2. /23 每个子网最多可容纳多少台可用主机？**

答案：

```text
2^9 - 2 = 510
```

**3. 某网络需要至少 36 个子网，每个子网约 800 台主机，最合适的掩码是：**

A. 255.255.248.0
B. 255.255.252.0
C. 255.255.254.0
D. 255.255.255.0

答案：B

---

## 8. 路由表最长前缀匹配

### 必须掌握

路由器转发分组时，采用**最长前缀匹配原则**。
如果没有任何路由匹配，则使用默认路由 Default。

判断方法：

```text
目的 IP 与子网掩码按位与
```

如果结果等于路由表中的网络地址，就匹配。

### 练习题

**1. 路由表中有如下表项：**

```text
10.1.0.0/16      接口 0
10.1.2.0/24      接口 1
Default          路由器 R
```

目的地址是：

```text
10.1.2.8
```

应该走哪条路由？

答案：接口 1，因为 /24 比 /16 更长。

**2. 路由表中有：**

```text
192.168.4.0/22   接口 0
192.168.8.0/22   接口 1
Default          路由器 R
```

目的地址：

```text
192.168.12.5
```

应该走哪条？

答案：Default，走路由器 R。

**3. /22 的地址块大小是多少个地址？**

答案：

```text
2^(32-22) = 1024 个地址
```

可用主机数为：

```text
1024 - 2 = 1022
```

## 9. RIP 距离向量路由协议

### 必须掌握

RIP 是距离向量协议，使用跳数作为度量。
RIP 最大跳数通常是 15，16 表示不可达。
收到邻居路由表后，新距离为：

```text
邻居通告距离 + 1
```

更新规则：

| 情况           | 处理         |
| ------------ | ---------- |
| 原来没有该网络      | 加入         |
| 新路径更短        | 更新         |
| 原下一跳就是该邻居    | 即使距离变大，也更新 |
| 新路径不更短且下一跳不同 | 不更新        |

### 练习题

**1. 路由器 A 收到邻居 C 通告：到 Net3 的距离为 2。A 到 C 还需要 1 跳。则 A 经 C 到 Net3 的距离是多少？**

答案：3

**2. A 原来到 Net5 的距离是 8，经 B。现在 C 通告到 Net5 的距离是 4。A 经 C 到 Net5 的新距离是多少？是否更新？**

答案：

```text
4 + 1 = 5
```

5 小于 8，更新，下一跳改为 C。

**3. RIP 使用什么作为路由度量？**

A. 带宽
B. 时延
C. 跳数
D. 丢包率

答案：C

## 10. OSPF 与 IGP

### 必须掌握

IGP 是内部网关协议，用于自治系统 AS 内部。
典型 IGP 包括 RIP 和 OSPF。
OSPF 是链路状态协议，使用 Dijkstra 最短路径算法。
OSPF 可划分区域，骨干区域是 Area 0。
连接多个区域的路由器叫 ABR。

### 练习题

**1. 下列哪些协议属于 IGP？**

A. RIP
B. OSPF
C. BGP
D. HTTP

答案：A、B

**2. OSPF 属于哪类路由协议？**

A. 距离向量协议
B. 链路状态协议
C. 应用层协议
D. 传输层协议

答案：B

**3. OSPF 中连接多个区域的路由器称为：**

A. DNS
B. ABR
C. DHCP
D. NAT

答案：B

## 11. 传输层复用与分用

### 必须掌握

复用：发送方多个应用进程共用传输层协议发送数据。
分用：接收方根据端口号把数据交给对应进程。
端口号是传输层的重要标识。

### 练习题

**1. 传输层复用发生在发送方还是接收方？**

答案：发送方。

**2. 传输层分用主要依据什么信息？**

A. MAC 地址
B. 端口号
C. 网卡型号
D. 路由协议

答案：B

**3. 多个应用程序同时使用 TCP 发送数据，这体现了传输层的什么功能？**

答案：复用。

## 12. Socket 套接字

### 必须掌握

Socket 是网络通信端点。
通常可以理解为：

```text
IP 地址 + 端口号
```

一个 TCP 连接更严格地由四元组标识：

```text
源 IP
源端口
目的 IP
目的端口
```

### 练习题

**1. Socket 通常由哪两部分组成？**

答案：

```text
IP 地址 + 端口号
```

**2. 一个 TCP 连接通常由什么四元组唯一标识？**

答案：

```text
源 IP、源端口、目的 IP、目的端口
```

**3. 端口号的主要作用是：**

A. 标识网络中的主机
B. 标识主机上的应用进程
C. 标识物理网卡
D. 标识交换机端口

答案：B

## 13. 文件传输时间计算

### 必须掌握

常见公式：

```text
RTT = 2 × 单向传播时延
```

```text
传输时延 = 数据量 / 带宽
```

```text
总时间 = 握手时延 + 传输时延 + 最后一位传播时延
```

如果题目说“握手延迟为 2 个 RTT”，则：

```text
握手时延 = 2 × RTT
```

### 练习题

**1. 单向传播时延为 30ms，则 RTT 是多少？**

答案：

```text
RTT = 2 × 30ms = 60ms
```

**2. 一个 80Mb 的文件，在 10Mbps 链路上传输，纯传输时延是多少？**

答案：

```text
80Mb / 10Mbps = 8s
```

**3. 若 RTT = 40ms，握手延迟为 2 个 RTT，则握手延迟是多少？**

答案：

```text
2 × 40ms = 80ms
```

## 14. 分组传输与 RTT 限制

### 必须掌握

如果题目规定“每个 RTT 只能发送若干个分组”，则要先算总分组数：

```text
总分组数 = 文件大小 / 分组大小
```

再算需要多少轮 RTT：

```text
RTT 轮数 = 总分组数 / 每 RTT 可发送分组数
```

注意单位：

```text
B = Byte = 字节
b = bit = 比特
1B = 8b
```

### 练习题

**1. 文件大小为 100KB，分组大小为 1KB，共需要多少个分组？**

答案：100 个分组。

**2. 总共有 500 个分组，每个 RTT 只能发送 10 个分组，至少需要多少个 RTT 轮次？**

答案：

```text
500 / 10 = 50 个 RTT
```

**3. 1MB 等于多少 bit？按 1MB = 10⁶ 字节计算。**

答案：

```text
1MB = 10^6 Byte = 8 × 10^6 bit
```

## 15. TCP Reno 拥塞控制

### 必须掌握

TCP Reno 在拥塞避免阶段：

```text
每经过 1 个 RTT，拥塞窗口 cwnd 增加 1 个 MSS
```

发生丢包后：

```text
cwnd 通常减半
```

最大窗口常用带宽时延积 BDP 计算：

```text
BDP = 带宽 × RTT
```

```text
最大窗口 = BDP / MSS
```

### 练习题

**1. 链路带宽为 100Mbps，RTT 为 50ms，BDP 是多少？**

答案：

```text
100 × 10^6 × 0.05 = 5 × 10^6 bit
```

**2. 若 BDP = 6 × 10⁶ bit，MSS = 1500 字节，则最大窗口是多少个报文段？**

答案：

```text
1500 字节 = 12000 bit
6 × 10^6 / 12000 = 500 个报文段
```

**3. TCP Reno 最大窗口为 600 个 MSS，丢包后减半。若每个 RTT 增加 1 个 MSS，要恢复到 600 个 MSS 需要多少个 RTT？**

答案：

```text
丢包后窗口 = 300
需要增加 = 600 - 300 = 300
所以需要 300 个 RTT
```

# 最终满分复习清单

你重点掌握下面这些，就能覆盖这套卷子的绝大部分考点：

```text
1. VLAN 是逻辑划分，与物理位置无关。
2. IEEE 802.1Q 会在以太网帧中加入 VLAN Tag。
3. ARP 请求广播，ARP 响应通常单播；ARP 不属于 UDP。
4. 802.11 使用 CSMA/CA，不是 CSMA/CD。
5. RTS/CTS 用于缓解隐藏终端问题。
6. CIDR /n 表示前 n 位是网络位，主机位是 32-n。
7. 路由聚合要找多个 IP 地址的最长共同前缀。
8. 子网划分要同时满足子网数和主机数。
9. 路由转发使用最长前缀匹配。
10. 没有匹配路由时走 Default 默认路由。
11. RIP 是距离向量协议，度量是跳数。
12. RIP 更新距离 = 邻居通告距离 + 1。
13. OSPF 是链路状态协议，使用 Dijkstra 算法。
14. IGP 是自治系统内部路由协议，包括 RIP、OSPF。
15. 传输层复用发生在发送方，分用发生在接收方。
16. Socket 通常是 IP 地址 + 端口号。
17. RTT = 2 × 单向传播时延。
18. 传输时延 = 数据量 / 带宽。
19. 总传输时间常见结构 = 握手时延 + 发送时延 + 传播时延。
20. BDP = 带宽 × RTT。
21. TCP 最大窗口 = BDP / MSS。
22. TCP Reno 丢包后窗口减半，拥塞避免阶段每 RTT 增加 1 MSS。
```

建议你优先刷这 5 类题：**CIDR 聚合、子网掩码、最长前缀匹配、RIP 更新、TCP/时延计算**。这些是这套卷子里最容易丢分、也最容易通过训练拿满分的部分。
