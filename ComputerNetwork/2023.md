## 3. 回退 N 帧协议

### 必须掌握

回退 N 帧协议，英文是：

```text
Go-Back-N，简称 GBN
```

核心特点：

```text
发送窗口可以大于 1
接收窗口固定为 1
出错后从出错帧开始全部重传
```

发送窗口最大值：

```text
Wmax = 2^n - 1
```

其中：

```text
n 是序号字段的位数
```

例如：

```text
序号字段 7 bit

Wmax = 2^7 - 1 = 127
```

### 练习题

1. 回退 N 帧协议中，如果序号字段为 5 bit，则发送窗口最大值是多少？

2. 回退 N 帧协议中，如果发送窗口最大值为 31，则至少需要多少位序号？

3. 判断题：回退 N 帧协议的接收窗口可以大于 1。对还是错？

参考答案：

```text
1. 2^5 - 1 = 31
2. 5 bit
3. 错。GBN 接收窗口固定为 1。
```

## 4. 选择重传协议

### 必须掌握

选择重传协议，英文是：

```text
Selective Repeat，简称 SR
```

核心特点：

```text
只重传出错或丢失的帧
接收方可以缓存乱序到达的帧
接收窗口可以大于 1
```

窗口大小通常满足：

```text
W ≤ 2^(n-1)
```

其中：

```text
n 是序号字段位数
```

### 练习题

1. 选择重传协议中，如果序号字段为 4 bit，则窗口最大值通常是多少？

2. 若选择重传协议需要窗口大小至少为 10，则至少需要多少位序号？

3. 选择题：选择重传协议相比回退 N 帧协议的主要优点是？

```text
A. 不需要确认
B. 不需要序号
C. 只重传出错帧
D. 接收窗口固定为 1
```

参考答案：

```text
1. 2^(4-1) = 8
2. 需要满足 10 ≤ 2^(n-1)，所以 n = 5
3. C
```

## 5. 滑动窗口与链路利用率计算

### 必须掌握

常见公式：

```text
帧发送时间 Tt = 帧长 / 带宽
```

为了使链路充分利用：

```text
W × Tt ≥ RTT + Tt
```

因此：

```text
W ≥ (RTT + Tt) / Tt
```

其中：

```text
W：窗口大小
Tt：发送一个数据帧的时间
RTT：往返传播时延
```

### 练习题

1. 一条链路带宽为 2 Mbps，帧长为 1000 Byte，求发送一个帧需要多长时间？

2. 一条链路 RTT 为 40 ms，帧发送时间为 5 ms，为保证链路充分利用，窗口大小至少是多少？

3. 带宽为 1 Mbps，RTT 为 30 ms，帧长为 1000 Byte，若采用回退 N 协议，至少需要多少位序号？

参考答案：

```text
1. 1000 Byte = 8000 bit
   Tt = 8000 / 2×10^6 = 0.004 s = 4 ms

2. W ≥ (40 + 5) / 5 = 9
   所以窗口至少为 9

3. 帧长 = 8000 bit
   Tt = 8000 / 1×10^6 = 8 ms

   W ≥ (30 + 8) / 8 = 4.75
   所以 W = 5

   回退 N：W ≤ 2^n - 1
   5 ≤ 2^n - 1
   n = 3
```

## 6. CSMA/CD 协议

### 必须掌握

CSMA/CD 是以太网中使用的介质访问控制协议。

全称：

```text
Carrier Sense Multiple Access with Collision Detection
载波监听多路访问 / 冲突检测
```

核心过程：

```text
先听后发
边发边听
冲突停止
随机退避
重新发送
```

### 练习题

1. CSMA/CD 中，“先听后发”是什么意思？

2. 判断题：CSMA/CD 协议可以完全避免冲突。对还是错？

3. 选择题：CSMA/CD 检测到冲突后，发送站应该怎么做？

```text
A. 继续发送
B. 立即停止发送，并随机退避后重传
C. 删除数据帧，不再发送
D. 改变目的地址
```

参考答案：

```text
1. 发送数据前先监听信道是否空闲。
2. 错。CSMA/CD 不能完全避免冲突，只能检测和处理冲突。
3. B
```

## 7. 二进制指数退避算法

### 必须掌握

二进制指数退避用于以太网发生冲突后的重传。

核心规律：

```text
冲突次数越多，退避时间范围越大
退避时间范围越大，再次冲突概率越低
```

所以：

```text
发生 4 次冲突的帧，比发生 1 次冲突的帧，再次冲突概率更低
```

### 练习题

1. 为什么二进制指数退避算法中，冲突次数越多，再次冲突概率越低？

2. 选择题：下列数据帧中，重传时再次发生冲突概率最低的是？

```text
A. 初次重传的帧
B. 发生两次冲突的帧
C. 发生三次冲突的帧
D. 发生四次冲突的帧
```

3. 判断题：二进制指数退避算法中，每次冲突后退避时间范围保持不变。对还是错？

参考答案：

```text
1. 因为可选择的随机退避时隙数增加，发送时间更加分散。
2. D
3. 错。退避时间范围会随冲突次数增加而扩大。
```

## 12. 子网掩码与子网地址计算

### 必须掌握

计算子网地址的方法：

```text
IP 地址 与 子网掩码 按位与
```

例如：

```text
IP：180.120.74.56
掩码：255.255.240.0

第三字节：
74  = 01001010
240 = 11110000
与   = 01000000 = 64
```

参考答案：

```text

   26  = 00011010
   240 = 11110000
   与  = 00010000 = 16

   子网地址：192.168.16.0

   37  = 00100101
```

## 16. 距离向量路由算法

### 必须掌握

距离向量路由算法核心思想：

```text
通过邻居告诉自己的距离向量，计算到各目的节点的最短距离
```

基本公式：

```text
D_C(X) = min { C 到邻居的代价 + 邻居到 X 的代价 }
```

做题步骤：

```text
1. 找出当前路由器的邻居
2. 写出到每个邻居的直接代价
3. 根据邻居发来的向量逐项计算
4. 取最小值
5. 确定下一跳
```

### 练习题

1. 路由器 C 到 A 的代价为 5，到 B 的代价为 8。
   A 到 D 的代价为 10，B 到 D 的代价为 4。
   C 到 D 的最短距离是多少？下一跳是谁？

2. 路由器 X 到邻居 Y 的代价为 3，到邻居 Z 的代价为 6。
   Y 到目标 N 的距离为 7，Z 到目标 N 的距离为 2。
   X 到 N 的最短距离是多少？下一跳是谁？

3. 判断题：距离向量路由算法中，路由器只需要知道邻居发来的距离向量，就可以更新自己的路由表。对还是错？

参考答案：

```text
1. 经 A：5 + 10 = 15
   经 B：8 + 4 = 12
   最短距离 12，下一跳 B

2. 经 Y：3 + 7 = 10
   经 Z：6 + 2 = 8
   最短距离 8，下一跳 Z

3. 对。
```

## 19. TCP 拥塞控制与慢启动

### 必须掌握

拥塞控制的目的：

```text
防止过多数据进入网络，造成网络拥塞
```

慢启动：

```text
TCP 连接刚开始时，拥塞窗口较小
每经过一个 RTT，拥塞窗口快速增大
通常表现为指数增长
达到慢启动阈值后，进入拥塞避免阶段
```

常见拥塞控制算法：

```text
慢启动
拥塞避免
快重传
快恢复
```

### 练习题

1. 什么是 TCP 慢启动？

2. 慢启动属于 TCP 的流量控制还是拥塞控制？

3. 判断题：慢启动阶段中，拥塞窗口一开始就设置得很大，以便尽快占满带宽。对还是错？

参考答案：

```text
1. 慢启动是 TCP 拥塞控制算法，开始时使用较小拥塞窗口，然后逐步增大。
2. 拥塞控制。
3. 错。慢启动开始时拥塞窗口较小。
```

## 20. 数字签名、加密、身份认证、防火墙

### 必须掌握

四者区别：

| 技术   | 主要作用          |
| ---- | ------------- |
| 数据加密 | 保证机密性，防止被别人看懂 |
| 数字签名 | 验证完整性、防篡改、防伪造 |
| 身份认证 | 验证对方身份        |
| 防火墙  | 访问控制，过滤非法流量   |

重点：

```text
验证报文是否被篡改和伪造：数字签名
```

### 练习题

1. 数字签名的主要作用是什么？

2. 数据加密和数字签名的区别是什么？

3. 选择题：通信接收方要验证收到的报文是否被篡改和伪造，应使用哪种技术？

```text
A. 数字签名
B. 数据加密
C. 防火墙
D. 身份认证
```

参考答案：

```text
1. 验证报文完整性、防篡改、防伪造。
2. 加密主要保证机密性；数字签名主要保证完整性和不可伪造性。
3. A
```
