# 计算机网络满分知识点汇总与练习题

## 1. 网络协议三要素

### 必须掌握

网络协议由三要素组成：

| 要素      | 含义            |
| ------- | ------------- |
| 语法      | 数据和控制信息的格式    |
| 语义      | 控制信息的含义、动作和响应 |
| 同步 / 时序 | 事件发生的顺序       |

### 练习题

1. 网络协议的三要素分别是什么？请分别解释。
2. 判断题：网络协议中语义和同步比语法更重要，语法可以忽略。对还是错？
3. TCP 三次握手体现了网络协议三要素中的哪一个要素？

### 答案速查

1. 语法、语义、同步。
2. 错。三者都重要。
3. 同步 / 时序。

## 2. 传播时延

### 必须掌握

```text
传播时延 = 链路长度 / 信号传播速度
```

传播时延只和 **链路长度**、**信号传播速度** 有关，和数据大小无关。

### 练习题

1. 某链路长度为 300 km，信号传播速度为 `2×10^5 km/s`，求传播时延。
2. 某信号从发送端到接收端需要 2 ms，传播速度为 `2×10^8 m/s`，求链路长度。
3. 判断题：帧越大，传播时延越大。对还是错？

### 答案速查

1. `300 / (2×10^5) = 0.0015 s = 1.5 ms`
2. `2×10^-3 × 2×10^8 = 4×10^5 m = 400 km`
3. 错。帧大小影响发送时延，不影响传播时延。

## 3. 发送时延 / 传输时延

### 必须掌握

```text
发送时延 = 数据长度 / 传输速率
数据长度 = 发送时延 × 传输速率
传输速率 = 数据长度 / 发送时延
```

常见单位换算：

```text
1 Byte = 8 bit
1 Mbps = 10^6 bit/s
1 ms = 10^-3 s
1 μs = 10^-6 s
```

### 练习题

1. 某信道速率为 100 Mbps，发送一个 20000 字节的数据帧，发送时延是多少？
2. 某信道速率为 10 Mbps，发送时延为 4 ms，求数据帧长度，单位为字节。
3. 一个 1500 字节的帧，在 6 Mbps 链路上传输，发送时延是多少？

### 答案速查

1. `20000×8 / 100×10^6 = 0.0016 s = 1.6 ms`
2. `10×10^6 × 4×10^-3 / 8 = 5000 字节`
3. `1500×8 / 6×10^6 = 0.002 s = 2 ms`

## 4. IP 地址与子网划分

### 必须掌握

核心公式：

```text
子网数 ≥ 2^子网位数
可用主机数 = 2^主机位数 - 2
前缀长度 = 原网络位 + 借用的子网位
```

常见掩码：

| 前缀  | 子网掩码            |
| --- | --------------- |
| /24 | 255.255.255.0   |
| /25 | 255.255.255.128 |
| /26 | 255.255.255.192 |
| /27 | 255.255.255.224 |
| /28 | 255.255.255.240 |
| /20 | 255.255.240.0   |

### 练习题

1. 某网络为 `192.168.10.0/24`，要求划分 4 个子网，每个子网至少支持 50 台主机，应该使用什么子网掩码？
2. 某网络为 `172.16.0.0/16`，要求划分 8 个子网，每个子网最多可容纳多少台主机？
3. `192.168.1.0/26` 的可用主机地址范围是多少？

### 答案速查

1. 需要 4 个子网，借 2 位，得到 `/26`，掩码 `255.255.255.192`，每个子网可用主机数 `62`。
2. 借 3 位，前缀为 `/19`，主机位 13 位，可用主机数 `2^13 - 2 = 8190`。
3. 网络地址 `192.168.1.0`，广播地址 `192.168.1.63`，可用范围 `192.168.1.1 ~ 192.168.1.62`。

## 5. C 类网络主机容量

### 必须掌握

C 类默认掩码：

```text
255.255.255.0 = /24
```

主机位 8 位，所以单个 C 类网络可用主机数：

```text
2^8 - 2 = 254
```

### 练习题

1. 若需要支持 600 台主机，至少需要几个 C 类网络？
2. 若需要支持 1000 台主机，至少需要几个 C 类网络？
3. 判断题：一个默认 C 类网络最多可以容纳 256 台可用主机。对还是错？

### 答案速查

1. `600 / 254 ≈ 2.36`，向上取整，需要 3 个。
2. `1000 / 254 ≈ 3.94`，向上取整，需要 4 个。
3. 错。可用主机数是 254，要排除网络地址和广播地址。

## 6. 数据链路层 ACK 与确认重传

### 必须掌握

数据链路层收到正确数据帧后，通常向发送方返回：

```text
ACK 确认帧
```

如果发送方超时未收到 ACK，则可能重传。

### 练习题

1. 数据链路层接收到正确的数据帧后，会向发送方返回什么信息？
2. 如果发送方在规定时间内没有收到 ACK，一般会采取什么操作？
3. ACK 的主要作用是什么？

### 答案速查

1. ACK。
2. 重传。
3. 告诉发送方数据已正确收到，用于可靠传输。

## 7. 滑动窗口协议：选择重传 SR

### 必须掌握

选择重传 SR 协议中：

```text
序号空间 = 2^n
最大接收窗口 = 2^(n-1)
```

其中 `n` 是序号字段的 bit 数。

### 练习题

1. 选择重传协议中，序号字段为 4 bit，接收窗口最大是多少？
2. 选择重传协议中，序号字段为 6 bit，接收窗口最大是多少？
3. 为什么选择重传协议的窗口不能大于序号空间的一半？

### 答案速查

1. `2^(4-1) = 8`
2. `2^(6-1) = 32`
3. 为了避免新帧和重传旧帧的序号混淆。

## 8. CSMA/CD 最大距离计算

### 必须掌握

CSMA/CD 要求发送方在发送完最小帧之前能够检测到冲突。

核心关系：

```text
2 × 传播时延 ≤ 最小帧发送时延
```

以太网最小帧长：

```text
64 字节 = 512 bit
```

### 练习题

1. 某以太网速率为 10 Mbps，传播速度为 200 m/μs，最小帧长为 512 bit，求最远站点距离。
2. 某以太网速率从 100 Mbps 提升到 1000 Mbps，若其他条件不变，为保证 CSMA/CD 正常工作，最大距离如何变化？
3. 为什么以太网速率越高，CSMA/CD 允许的最大网络直径越小？

### 答案速查

1. 发送时延 `512 / 10Mbps = 51.2 μs`，单程传播时延 `25.6 μs`，距离 `200×25.6 = 5120 m`。
2. 缩短为原来的 `1/10`。
3. 速率越高，最小帧发送时间越短，留给冲突检测的传播时间更短，所以距离必须缩短。

## 9. CSMA/CA 与隐蔽站问题

### 必须掌握

无线局域网使用：

```text
CSMA/CA
```

解决隐蔽站问题的机制是：

```text
RTS/CTS
```

区别：

| 协议      | 场景    | 核心思想 |
| ------- | ----- | ---- |
| CSMA/CD | 有线以太网 | 冲突检测 |
| CSMA/CA | 无线局域网 | 冲突避免 |

### 练习题

1. 无线局域网中，解决隐蔽站问题的机制是什么？
2. CSMA/CD 和 CSMA/CA 分别主要用于什么网络？
3. 为什么无线局域网通常不用 CSMA/CD？

### 答案速查

1. RTS/CTS。
2. CSMA/CD 用于有线以太网；CSMA/CA 用于无线局域网。
3. 无线环境中难以边发送边检测冲突，因此采用冲突避免。

## 10. 路由器工作原理

### 必须掌握

路由器：

```text
工作在网络层
根据 IP 地址转发分组
根据路由表选择路径
```

设备层次对比：

| 设备  | 层次    | 转发依据   |
| --- | ----- | ------ |
| 集线器 | 物理层   | 比特信号   |
| 交换机 | 数据链路层 | MAC 地址 |
| 路由器 | 网络层   | IP 地址  |

### 练习题

1. 路由器工作在 OSI 参考模型的哪一层？
2. 路由器转发 IP 分组时主要依据什么？
3. 判断题：路由器根据 MAC 地址转发数据帧。对还是错？

### 答案速查

1. 网络层。
2. 目标 IP 地址和路由表。
3. 错。交换机根据 MAC 地址转发，路由器根据 IP 地址转发。

## 11. OSPF 与链路状态路由

### 必须掌握

OSPF 是：

```text
链路状态路由协议
```

核心机制：

```text
通过泛洪 Flooding 传播 LSA
```

核心概念：

| 概念       | 含义      |
| -------- | ------- |
| LSA      | 链路状态通告  |
| LSDB     | 链路状态数据库 |
| Flooding | 泛洪      |
| Dijkstra | 最短路径算法  |

### 练习题

1. OSPF 通过什么机制在自治系统内传播 LSA？
2. OSPF 属于距离向量路由协议还是链路状态路由协议？
3. OSPF 路由器基于 LSDB 使用什么算法计算最短路径？

### 答案速查

1. 泛洪 Flooding。
2. 链路状态路由协议。
3. Dijkstra 算法。

## 13. 停止-等待协议利用率

### 必须掌握

忽略 ACK 长度时：

```text
信道利用率 U = 发送时延 / (发送时延 + 2 × 传播时延)
```

提高利用率的方法：

```text
1. 使用滑动窗口协议
2. 增大帧长
3. 减少确认等待开销
```

### 练习题

1. 某链路传播时延为 1 ms，帧发送时延为 1 ms，停止-等待协议的信道利用率是多少？
2. 某链路传播时延为 2 ms，帧发送时延为 0.5 ms，停止-等待协议的信道利用率是多少？
3. 写出两种提高停止-等待协议信道利用率的方法。

### 答案速查

1. `1 / (1 + 2×1) = 1/3 ≈ 33.3%`
2. `0.5 / (0.5 + 2×2) = 0.5 / 4.5 ≈ 11.1%`
3. 使用滑动窗口协议；增大帧长；减少确认开销。

## 14. 网络安全：防止网络嗅探

### 必须掌握

网络嗅探攻击的核心是窃听网络中传输的数据。

有效防范方法：

```text
加密传输
```

例如：

```text
HTTPS
TLS / SSL
SSH
VPN
```

### 练习题

1. 防止网络嗅探攻击最有效的措施是什么？
2. 判断题：设置复杂密码可以防止明文数据被嗅探。对还是错？
3. 为什么加密传输可以防止嗅探攻击？

### 答案速查

1. 对网络数据进行加密传输。
2. 错。复杂密码防暴力破解，但明文传输仍可能被抓包。
3. 即使攻击者捕获数据，也无法直接解读密文内容。
