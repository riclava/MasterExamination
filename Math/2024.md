# 同等学力计算机综合数学基础：知识点与练习题汇总

## 1. 谓词逻辑与量词符号化

### 必须掌握

要会把自然语言翻译成逻辑符号。

重点句型：

```text
所有……：∀x
存在……：∃x
不是所有……：¬∀x，等价于 ∃x¬
没有……：¬∃x，等价于 ∀x¬
只有 A 才 B：B → A
A 是 B 的必要条件：B → A
A 是 B 的充分条件：A → B
```

### 练习题

1. 设 `S(x)` 表示“x 是学生”，`P(x)` 表示“x 通过考试”，`G(x)` 表示“x 可以毕业”。
   用谓词逻辑表示：
   **只有通过考试的学生才能毕业。**

2. 设 `M(x)` 表示“x 是人”，`R(x)` 表示“x 努力”，`S(x)` 表示“x 成功”。
   用谓词逻辑表示：
   **不是所有努力的人都会成功。**

## 2. 命题联结词与完备集

### 必须掌握

常见联结词：

```text
¬：非
∧：且
∨：或
→：蕴含
↔：等价
```

常见完备集：

```text
{¬, ∧, ∨}
{¬, ∧}
{¬, ∨}
{¬, →}
```

不是完备集的典型例子：

```text
{∧, ∨}
```

因为没有否定，无法表达“非”。

### 练习题

1. 判断 `{¬, ∧}` 是否是联结词完备集，并说明理由。

2. 判断 `{∧, ∨}` 是否是联结词完备集，并说明理由。

## 3. 集合、空集、属于与包含

### 必须掌握

最容易混淆的是：

```text
∅：空集
{∅}：含有空集这个元素的集合
{{∅}}：含有 {∅} 这个元素的集合
```

以及：

```text
a ∈ A：a 是 A 的元素
A ⊆ B：A 是 B 的子集
```

### 练习题

1. 设 `A = {∅, {∅}}`，判断下列命题真假：

```text
∅ ∈ A
{∅} ∈ A
∅ ⊆ A
{{∅}} ⊆ A
```

2. 设 `A = {1, {1}, ∅}`，判断下列命题真假：

```text
1 ∈ A
{1} ∈ A
{1} ⊆ A
∅ ∈ A
∅ ⊆ A
```

## 4. 幂集

### 必须掌握

幂集是一个集合所有子集组成的集合。

```text
P(A) = A 的所有子集构成的集合
如果 |A| = n，则 |P(A)| = 2^n
```

特别注意：幂集的元素本身都是集合。

### 练习题

1. 设 `A = {a, b}`，求：

```text
P(A)
P(P(A)) 的元素个数
```

2. 设 `A = {∅}`，求：

```text
P(A)
P(P(A))
```

## 5. 多项式定理与项的系数

### 必须掌握

对于：

```text
(x1 + x2 + ... + xr)^n
```

项：

```text
x1^a1 x2^a2 ... xr^ar
```

的系数为：

```text
n! / (a1! a2! ... ar!)
```

其中：

```text
a1 + a2 + ... + ar = n
```

### 练习题

1. 求 `(x + y + z)^6` 中 `x^2 y^3 z` 的系数。

2. 求 `(a + b + c + d)^5` 中 `a^2 b c d` 的系数。

## 6. 排列组合：男女相间与位置限制

### 必须掌握

这类题的核心是：

```text
先确定位置模式
再分别排列不同类型的人或物
```

例如男女相间：

```text
男 女 男 女 ...
或
女 男 女 男 ...
```

如果还规定最后一位性别，就会唯一确定位置模式。

### 练习题

1. 有 4 个男生和 4 个女生排成一排，要求男女相间且女生站在最后一位，问有多少种排法？

2. 有 5 个男生和 5 个女生排成一排，要求男女相间且男生站在第一位，问有多少种排法？

## 7. 隔板法：非负整数解与正整数解

### 必须掌握

非负整数解：

```text
x1 + x2 + ... + xk = n, xi ≥ 0

解数 = C(n + k - 1, k - 1)
```

正整数解：

```text
x1 + x2 + ... + xk = n, xi ≥ 1

解数 = C(n - 1, k - 1)
```

### 练习题

1. 求方程：

```text
x1 + x2 + x3 + x4 = 10
```

的非负整数解个数。

2. 求方程：

```text
x1 + x2 + x3 + x4 = 10
```

的正整数解个数。

## 8. 容斥原理与相邻块法

### 必须掌握

容斥原理：

```text
不满足 A 和 B 的数量
= 总数 - 满足 A 的数量 - 满足 B 的数量 + 同时满足 A 和 B 的数量
```

相邻块法：

```text
如果要求 abc 连续且顺序固定，则把 abc 看成一个整体
```

### 练习题

1. `a, b, c, d, e` 五个元素全排列中，不允许出现连续子串 `ab`，有多少种排列？

2. `a, b, c, d, e, f` 六个元素全排列中，不允许出现连续子串 `ab` 和 `cd`，有多少种排列？

## 9. 二元关系与传递闭包

### 必须掌握

传递闭包就是把关系中所有能通过“路径”推出的有序对补上。

如果：

```text
<a, b> ∈ R
<b, c> ∈ R
```

则传递闭包中必须有：

```text
<a, c>
```

### 练习题

1. 设 `A = {1, 2, 3, 4}`，`R = {<1,2>, <2,4>}`，求 `t(R)`。

2. 设 `A = {1, 2, 3, 4, 5}`，`R = {<1,2>, <2,3>, <3,5>}`，求 `t(R)` 中有多少个有序对。

## 10. 偏序关系、极大元、极小元、链与反链

### 必须掌握

偏序关系满足：

```text
自反性
反对称性
传递性
```

重要概念：

```text
极大元：没有比它更大的元素
最大元：比所有元素都大

极小元：没有比它更小的元素
最小元：比所有元素都小

链：任意两个元素都可比
反链：任意两个元素都不可比
```

注意：

```text
极大元不一定是最大元
极小元不一定是最小元
```

### 练习题

1. 设 `A = {2, 3, 4, 6, 8, 12}`，关系为整除关系。求：

```text
极大元集合
极小元集合
最大元
最小元
```

2. 设 `A = {2, 3, 5, 6, 10, 15, 30}`，关系为整除关系。求：

```text
最长链长度
一个最长链
极大元集合
极小元集合
```

## 11. 图的邻接矩阵

### 必须掌握

如果无向图有 `n` 个顶点，那么邻接矩阵一定是：

```text
n × n
```

与边数 `m` 无关。

边数只影响矩阵中 1 的数量，不影响矩阵维度。

### 练习题

1. 一个无向图有 8 个顶点、12 条边，它的邻接矩阵是几行几列？

2. 一个有向图有 6 个顶点、10 条边，它的邻接矩阵是几行几列？

## 12. 二叉树节点数性质

### 必须掌握

设：

```text
n0 = 叶子节点数
n1 = 度为 1 的节点数
n2 = 度为 2 的节点数
```

二叉树中恒有：

```text
n0 = n2 + 1
```

所以：

```text
n2 = n0 - 1
```

### 练习题

1. 一棵二叉树有 10 个叶子节点，求度为 2 的节点个数。

2. 一棵二叉树中度为 2 的节点有 15 个，求叶子节点个数。

## 13. 一阶线性递推关系

### 必须掌握

常见形式：

```text
an = p·a(n-1) + q
```

解题步骤：

```text
1. 解齐次递推：an = p·a(n-1)
2. 找一个常数特解
3. 写出通解
4. 代入初值求常数
5. 得到 an
```

### 练习题

1. 已知：

```text
an = 3a(n-1) + 2, a0 = 1
```

求 `a3`。

2. 已知：

```text
an = 2a(n-1) - 5, a0 = 3
```

求通项公式 `an`。

## 14. 集合证明：对称差与笛卡尔积

### 必须掌握

对称差：

```text
A ⊕ B = (A - B) ∪ (B - A)
```

笛卡尔积：

```text
A × C = { (x, y) | x ∈ A 且 y ∈ C }
```

证明集合相等的标准方法：

```text
先证左边 ⊆ 右边
再证右边 ⊆ 左边
```

### 练习题

1. 证明：

```text
A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
```

2. 证明：

```text
(A - B) × C = (A × C) - (B × C)
```

## 15. 平面图、面度数与握手定理

### 必须掌握

平面图中：

```text
所有面的度数之和 = 2m
```

其中 `m` 是边数。

如果每个面的度数都不少于 `k`，则：

```text
kf ≤ 2m
```

所以：

```text
f ≤ 2m / k
```

本卷中每个面的度数不少于 4，所以：

```text
4f ≤ 2m
f ≤ m / 2
```

### 练习题

1. 设简单连通平面图每个面的度数都不少于 5，边数为 `m`，面数为 `f`。证明：

```text
f ≤ 2m / 5
```

2. 设一个连通平面图有 12 条边，且每个面的度数都不少于 4，问面数最多是多少？

## 2. 联结词完备集

1. `{¬, ∧}` 是完备集，因为可以表示：

```text
A ∨ B = ¬(¬A ∧ ¬B)
```

2. `{∧, ∨}` 不是完备集，因为无法表示否定 `¬`。

## 13. 递推关系

1. `an = 3a(n-1) + 2, a0 = 1`：

```text
a1 = 5
a2 = 17
a3 = 53
```

2. `an = 2a(n-1) - 5, a0 = 3`

设常数特解为 `A`：

```text
A = 2A - 5
A = 5
```

通解：

```text
an = C·2^n + 5
```

代入 `a0 = 3`：

```text
C + 5 = 3
C = -2
```

所以：

```text
an = 5 - 2^(n+1)
```

## 14. 集合证明

这两题都用“双向包含法”证明。

重点写法：

```text
任取 x ∈ 左边，推出 x ∈ 右边；
任取 x ∈ 右边，推出 x ∈ 左边。
```

## 15. 平面图

1. 每个面的度数不少于 5：

```text

∑deg(F) = 2m
所以 5f ≤ 2m
所以 f ≤ 2m/5
```

2. `m = 12`，每个面的度数不少于 4：

```text
f ≤ m/2 = 6
```

所以面数最多为：

```text
6
```
