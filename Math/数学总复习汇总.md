# 二、逻辑

## 2.1 谓词逻辑符号化

### 必须掌握

常用符号：

```text
∀：任意、所有、每一个
∃：存在、至少一个
∃!：存在唯一
¬：非、不是
∧：并且
∨：或者
→：如果……那么……
↔：当且仅当
```

常见翻译模板：

```text
所有 A 都是 B：∀x(A(x) → B(x))
有些 A 是 B：∃x(A(x) ∧ B(x))
有些 A 不是 B：∃x(A(x) ∧ ¬B(x))
不是所有 A 都是 B：¬∀x(A(x) → B(x))，等价于 ∃x(A(x) ∧ ¬B(x))
没有 A 是 B：¬∃x(A(x) ∧ B(x))，等价于 ∀x(A(x) → ¬B(x))
只有 A 才 B：B → A
A 是 B 的充分条件：A → B
A 是 B 的必要条件：B → A
任意两个不同对象不满足 R：∀x∀y((A(x) ∧ A(y) ∧ x≠y) → ¬R(x,y))
```

唯一存在：

```text
∃!y P(y)
```

可展开为：

```text
∃y(P(y) ∧ ∀z(P(z) → z=y))
```

### 解题技巧

1. 先找主语范围，再写成 `A(x) → ...`。
2. “所有/任意”通常用蕴含；“存在/有些”通常用合取。
3. “只有 A 才 B”不要写反，应写 `B → A`。
4. “不是所有”优先改成“存在一个反例”。

### 练习题与答案

**题 1：** 用谓词逻辑表示：所有学生都参加考试。设 `S(x)`：x 是学生，`E(x)`：x 参加考试。

答案：

```text
∀x(S(x) → E(x))
```

**题 2：** 用谓词逻辑表示：有些学生没有参加考试。

答案：

```text
∃x(S(x) ∧ ¬E(x))
```

**题 3：** 用谓词逻辑表示：只有通过考试的学生才能毕业。设 `S(x)`：x 是学生，`P(x)`：x 通过考试，`G(x)`：x 可以毕业。

答案：

```text
∀x((S(x) ∧ G(x)) → P(x))
```

也可写作：

```text
∀x(S(x) → (G(x) → P(x)))
```

**题 4：** 用谓词逻辑表示：每个自然数都有唯一后继。设 `N(x)`：x 是自然数，`S(x,y)`：y 是 x 的后继。

答案：

```text
∀x(N(x) → ∃!y(N(y) ∧ S(x,y)))
```

展开为：

```text
∀x(N(x) → ∃y(N(y) ∧ S(x,y) ∧ ∀z((N(z) ∧ S(x,z)) → z=y)))
```

**题 5：** 用谓词逻辑表示：没有两个不同的人拥有相同身份证号。设 `P(x)`：x 是人，`I(x,y)`：x 和 y 身份证号相同。

答案：

```text
∀x∀y((P(x) ∧ P(y) ∧ x≠y) → ¬I(x,y))
```

## 2.2 量词展开与量词否定

### 必须掌握

有限个体域：

```text
D = {a,b,c}

∀xP(x) = P(a) ∧ P(b) ∧ P(c)
∃xP(x) = P(a) ∨ P(b) ∨ P(c)
```

量词否定：

```text
¬∀xP(x) ≡ ∃x¬P(x)
¬∃xP(x) ≡ ∀x¬P(x)
```

### 解题技巧

1. 全称量词展开成“并且”。
2. 存在量词展开成“或者”。
3. 否定进量词时，量词互换，谓词取否定。

### 练习题与答案

**题 1：** 个体域为 `{a,b,c}`，展开 `∀xP(x)`。

答案：

```text
P(a) ∧ P(b) ∧ P(c)
```

**题 2：** 个体域为 `{1,2,3}`，展开 `∃xQ(x)`。

答案：

```text
Q(1) ∨ Q(2) ∨ Q(3)
```

**题 3：** 化简 `¬(∀xP(x))`。

答案：

```text
∃x¬P(x)
```

**题 4：** 化简 `¬(∃x(P(x) ∧ Q(x)))`。

答案：

```text
∀x(¬P(x) ∨ ¬Q(x))
```

## 2.3 命题逻辑等价、真值表、范式

### 必须掌握

核心等价式：

```text
P → Q ≡ ¬P ∨ Q
¬(P → Q) ≡ P ∧ ¬Q
¬(P ∧ Q) ≡ ¬P ∨ ¬Q
¬(P ∨ Q) ≡ ¬P ∧ ¬Q
P ↔ Q ≡ (P → Q) ∧ (Q → P)
```

蕴含式真值：

```text
P → Q 只有 P=1 且 Q=0 时为假，其余都为真。
```

主析取范式：

```text
取真值为 1 的行，用极小项 m_i 表示，用 ∨ 连接。
```

主合取范式：

```text
取真值为 0 的行，用极大项 M_i 表示，用 ∧ 连接。
```

三变量编号顺序：

| 编号 | P | Q | R |
| --- | --- | --- | --- |
| 0 | 0 | 0 | 0 |
| 1 | 0 | 0 | 1 |
| 2 | 0 | 1 | 0 |
| 3 | 0 | 1 | 1 |
| 4 | 1 | 0 | 0 |
| 5 | 1 | 0 | 1 |
| 6 | 1 | 1 | 0 |
| 7 | 1 | 1 | 1 |

### 解题技巧

1. 看到蕴含，先改写成 `¬P ∨ Q`。
2. 求主析取/主合取，先画真值表，再按编号写。
3. 判断矛盾前提，先化简否定与蕴含。

### 练习题与答案

**题 1：** 化简 `¬(P → Q)`。

答案：

```text
P ∧ ¬Q
```

**题 2：** 化简 `¬(P ∨ Q) ∨ P`。

答案：

```text
(¬P ∧ ¬Q) ∨ P ≡ P ∨ ¬Q
```

**题 3：** 判断前提 `¬(P ∨ Q), P` 是否矛盾。

答案：

```text
¬(P ∨ Q) ≡ ¬P ∧ ¬Q
```

可推出 `¬P`，但前提又有 `P`，所以矛盾。

**题 4：** 求 `P → Q` 的主析取范式编号。

答案：

两变量编号顺序 `(P,Q)`：

```text
0: 00
1: 01
2: 10
3: 11
```

`P → Q` 只在 `10` 为假，所以主析取范式为：

```text
m0 ∨ m1 ∨ m3
```

主合取范式为：

```text
M2
```

## 2.4 联结词完备集、NAND 与 NOR

### 必须掌握

常见完备集：

```text
{¬, ∧, ∨}
{¬, ∧}
{¬, ∨}
{¬, →}
```

不是完备集的典型例子：

```text
{∧, ∨}
```

因为没有否定。

与非 NAND：

```text
P ↑ Q = ¬(P ∧ Q)

¬P = P ↑ P
P ∧ Q = (P ↑ Q) ↑ (P ↑ Q)
P ∨ Q = (P ↑ P) ↑ (Q ↑ Q)
P → Q = P ↑ (Q ↑ Q)
```

或非 NOR：

```text
P ↓ Q = ¬(P ∨ Q)

¬P = P ↓ P
P ∨ Q = (P ↓ Q) ↓ (P ↓ Q)
P ∧ Q = (P ↓ P) ↓ (Q ↓ Q)
```

### 练习题与答案

**题 1：** 判断 `{¬, ∧}` 是否是联结词完备集。

答案：

是。因为：

```text
A ∨ B = ¬(¬A ∧ ¬B)
```

**题 2：** 判断 `{∧, ∨}` 是否是联结词完备集。

答案：

不是。它不能表达否定 `¬`。

**题 3：** 已知 `P ↑ Q = ¬(P ∧ Q)`，只用 `↑` 表示 `P ∨ Q`。

答案：

```text
(P ↑ P) ↑ (Q ↑ Q)
```

**题 4：** 已知 `P ↓ Q = ¬(P ∨ Q)`，只用 `↓` 表示 `P ∧ Q`。

答案：

```text
(P ↓ P) ↓ (Q ↓ Q)
```

# 三、集合与关系

## 3.1 集合基础、幂集、集合运算

### 必须掌握

易混概念：

```text
∅：空集
{∅}：含有空集这个元素的集合
{{∅}}：含有 {∅} 这个元素的集合

a ∈ A：a 是 A 的元素
A ⊆ B：A 是 B 的子集
```

幂集：

```text
P(A) = A 的所有子集组成的集合
若 |A| = n，则 |P(A)| = 2^n
```

集合差与对称差：

```text
A - B = {x | x∈A 且 x∉B} = A ∩ B^c
A ⊕ B = (A - B) ∪ (B - A)
A ⊕ B = (A ∪ B) - (A ∩ B)
```

子集计数：

```text
n 元集合子集总数 = 2^n
非空子集数 = 2^n - 1
真子集数 = 2^n - 1
非空真子集数 = 2^n - 2
奇数个元素的子集数 = 2^(n-1)
偶数个元素的子集数 = 2^(n-1)
```

### 解题技巧

1. 判断 `∈` 和 `⊆` 时，把对象层级看清。
2. 幂集的元素一定是集合。
3. 证明集合相等，标准方法是“双向包含法”或“元素等价变形”。

### 练习题与答案

**题 1：** 设 `A = {∅, {∅}}`，判断：

```text
∅ ∈ A
{∅} ∈ A
∅ ⊆ A
{{∅}} ⊆ A
```

答案：

```text
∅ ∈ A：真
{∅} ∈ A：真
∅ ⊆ A：真
{{∅}} ⊆ A：真
```

**题 2：** 设 `A = {a,b}`，求 `P(A)` 和 `|P(P(A))|`。

答案：

```text
P(A) = {∅, {a}, {b}, {a,b}}
|P(P(A))| = 2^4 = 16
```

**题 3：** 一个集合有 15 个元素，求非空真子集个数。

答案：

```text
2^15 - 2 = 32766
```

**题 4：** 设 `A={1,2,3,4}`，`B={3,4,5}`，求 `A ⊕ B`。

答案：

```text
A ⊕ B = {1,2,5}
```

## 3.2 二元关系、逆关系、复合关系、关系数量

### 必须掌握

二元关系：

```text
R ⊆ A × A
若 |A| = n，则 |A × A| = n^2
A 上二元关系总数 = 2^(n^2)
```

逆关系：

```text
R^{-1} = {(y,x) | (x,y)∈R}
```

复合关系：

```text
R ∘ S 表示先走 S，再走 R
x(S)z 且 z(R)y，则 x(R∘S)y
```

关系幂：

```text
R^0 = I_A
R^1 = R
R^2 = R ∘ R
R^(n+1) = R^n ∘ R
```

关系的像：

```text
R[S] = {y | ∃x∈S，使得 (x,y)∈R}
```

传递闭包：

```text
如果 (a,b)∈R 且 (b,c)∈R，则 t(R) 中要补上 (a,c)。
```

### 解题技巧

1. 复合关系先看右边关系，再看左边关系。
2. 传递闭包可把关系看成有向路径，能走到就补有序对。
3. 关系数量题只看 `A×A` 的大小。

### 练习题与答案

**题 1：** 若集合 `A` 有 4 个元素，求 `A×A` 的元素数和 A 上二元关系总数。

答案：

```text
|A×A| = 4^2 = 16
二元关系总数 = 2^16 = 65536
```

**题 2：** 设 `R={(1,2),(2,3),(3,4)}`，`S={(2,5),(3,6),(4,7)}`，求 `S∘R`。

答案：

先走 `R`，再走 `S`：

```text
1 R 2, 2 S 5，所以 (1,5)
2 R 3, 3 S 6，所以 (2,6)
3 R 4, 4 S 7，所以 (3,7)

S∘R = {(1,5),(2,6),(3,7)}
```

**题 3：** 设 `A={1,2,3,4}`，`R={(1,2),(2,4)}`，求 `t(R)`。

答案：

```text
t(R) = {(1,2),(2,4),(1,4)}
```

**题 4：** 设关系 `R={(x,y) | y=x^2}`，`A={-1,0,1,2}`，求 `R[A]`。

答案：

```text
R[A] = {0,1,4}
```

## 3.3 等价关系、划分、商集

### 必须掌握

等价关系满足：

```text
自反性：∀x∈A, xRx
对称性：若 xRy，则 yRx
传递性：若 xRy 且 yRz，则 xRz
```

等价关系与划分一一对应。

若等价类大小为：

```text
n1, n2, ..., nk
```

则对应等价关系中的有序对个数为：

```text
n1^2 + n2^2 + ... + nk^2
```

划分要求：

```text
1. 每个子集非空
2. 子集两两不交
3. 所有子集并起来等于原集合
```

### 解题技巧

1. 判断等价关系，逐条检查自反、对称、传递。
2. 给定划分写关系：同一块内元素两两相关。
3. 求有序对数量：每个等价类大小平方后相加。

### 练习题与答案

**题 1：** 集合 `A={1,2,3,4,5}` 划分为 `π={{1,2},{3},{4,5}}`，求对应等价关系中有多少个有序对。

答案：

```text
2^2 + 1^2 + 2^2 = 9
```

**题 2：** 判断 `{{1,2},{2,3},{4}}` 是否是 `{1,2,3,4}` 的划分。

答案：

不是。因为 `{1,2}` 与 `{2,3}` 有交集 `2`，不满足两两不交。

**题 3：** 在整数集 `Z` 上定义 `aRb ⇔ a-b` 是偶数，判断 R 是否为等价关系。

答案：

是。

```text
自反：a-a=0，是偶数。
对称：若 a-b 是偶数，则 b-a 也是偶数。
传递：若 a-b、b-c 都是偶数，则 a-c=(a-b)+(b-c) 也是偶数。
```

**题 4：** 在 `Z` 上定义 `aRb ⇔ a≡b (mod 3)`，求商集。

答案：

```text
Z/R = {[0], [1], [2]}
```

其中：

```text
[0] = {...,-6,-3,0,3,6,...}
[1] = {...,-5,-2,1,4,7,...}
[2] = {...,-4,-1,2,5,8,...}
```

## 3.4 偏序关系、链、反链、极值元素

### 必须掌握

偏序关系满足：

```text
自反性
反对称性
传递性
```

反对称性：

```text
若 aRb 且 bRa，则 a=b。
```

重要概念：

```text
极大元：没有比它更大的元素
最大元：比所有元素都大
极小元：没有比它更小的元素
最小元：比所有元素都小
链：任意两个元素都可比
反链：任意两个不同元素都不可比
```

典型偏序：

```text
整除关系 a|b
包含关系 A⊆B
小于等于关系 a≤b
```

### 解题技巧

1. 整除偏序中，`a≤b` 可理解为 `a|b`。
2. 极小元不是最小元，极大元不是最大元。
3. 最大元如果存在，一定唯一；极大元可以有多个。

### 练习题与答案

**题 1：** 设 `A={2,3,4,6,8,12}`，关系为整除关系，求极小元、极大元、最大元、最小元。

答案：

```text
极小元：{2,3}
极大元：{8,12}
最大元：无
最小元：无
```

**题 2：** 设 `A={1,2,3,4,6,12}`，在整除关系下写出一条最长链。

答案：

```text
1 | 2 | 4 | 12
```

或：

```text
1 | 2 | 6 | 12
1 | 3 | 6 | 12
```

最长链长度为 `4`。

**题 3：** 设 `A={1,2,3,4,5,6}`，判断 `{4,5,6}` 是否为反链。

答案：

是。因为任意两个不同元素之间都不存在整除关系。

# 四、排列组合与计数

## 4.1 基本计数、排列、组合、函数计数

### 必须掌握

基本原理：

```text
分类完成一件事：加法原理
分步完成一件事：乘法原理
```

排列组合：

```text
P(n,k) = n! / (n-k)!
C(n,k) = n! / (k!(n-k)!)
```

函数计数，设 `|A|=m`，`|B|=n`，`f:A→B`：

```text
函数总数：n^m
单射存在条件：m≤n，个数 n!/(n-m)!
双射存在条件：m=n，个数 n!
满射存在条件：m≥n，个数 ∑_{k=0}^n (-1)^k C(n,k)(n-k)^m
```

小规模满射可用：

```text
总函数数 - 非满射数
```

### 解题技巧

1. 选人无职位：组合。
2. 选人有职位：排列。
3. 分到固定组：组有名字，不除以组排列。
4. 分成无序组：组无名字，要除以组之间的排列数。

### 练习题与答案

**题 1：** 从 10 名学生中选 3 人参加比赛，有多少种选法？

答案：

```text
C(10,3) = 120
```

**题 2：** 从 10 名学生中选 3 人分别担任队长、副队长、记录员，有多少种方法？

答案：

```text
P(10,3) = 720
```

**题 3：** 设 `|A|=3`，`|B|=5`，从 A 到 B 的单射函数有多少个？

答案：

```text
5×4×3 = 60
```

**题 4：** 设 `|A|=3`，`|B|=2`，从 A 到 B 的满射函数有多少个？

答案：

```text
2^3 - 2 = 6
```

## 4.2 特殊排列：重复元素、插空、男女相间、圆排列

### 必须掌握

重复元素排列：

```text
n 个元素中有 a1,a2,...,ak 个重复：
不同排列数 = n!/(a1!a2!...ak!)
```

男女相间：

```text
男女人数相等：两种模式，男先或女先
总数 = 2 × m! × m!
```

插空法：

```text
先排不受限制的一类，再把受限制对象插入空位。
```

圆排列：

```text
n 个人围成一圈：(n-1)!
```

相邻块法：

```text
要求若干对象相邻，就捆成一个整体。
```

### 解题技巧

1. “不能相邻”优先用插空法或总数减相邻。
2. “必须相邻”优先用捆绑法。
3. 圆排列先固定旋转，再排列。
4. 附加条件如“最后一位是女生”会减少位置模式。

### 练习题与答案

**题 1：** 字母 `a,a,b,c` 能组成多少个不同排列？

答案：

```text
4!/2! = 12
```

**题 2：** 4 名男生和 4 名女生排成一排，要求男女交替排列，共有多少种？

答案：

```text
2 × 4! × 4! = 1152
```

**题 3：** 5 名男生和 4 名女生排成一排，要求女生互不相邻，共有多少种？

答案：

先排男生：

```text
5!
```

形成 6 个空位，选 4 个放女生并排列：

```text
C(6,4) × 4!
```

总数：

```text
5! × C(6,4) × 4! = 43200
```

**题 4：** 6 个人围成一圈，有多少种不同坐法？

答案：

```text
(6-1)! = 120
```

**题 5：** 5 个不同字母 A、B、C、D、E 排成一行，要求 A、B 不相邻，有多少种？

答案：

```text
总排列：5! = 120
A、B 相邻：4! × 2 = 48
不相邻：120 - 48 = 72
```

## 4.3 分组、盒子、隔板法

### 必须掌握

不同物品与盒子：

| 物品 | 盒子 | 常用方法 |
| --- | --- | --- |
| 不同 | 不同 | 函数/分配 |
| 不同 | 相同 | 集合划分 |
| 相同 | 不同 | 隔板法 |
| 相同 | 相同 | 整数拆分 |

无序分组：

```text
6 人分成 3 个无序小组，每组 2 人：
6!/(2!2!2!3!)
```

隔板法：

```text
x1+x2+...+xk = n, xi≥0
解数 = C(n+k-1, k-1)

x1+x2+...+xk = n, xi≥1
解数 = C(n-1, k-1)
```

### 解题技巧

1. 盒子有名字，不除以盒子排列。
2. 盒子无名字，转化为划分或除以组间排列。
3. 正整数解先每个变量减 1，再用非负整数解公式。

### 练习题与答案

**题 1：** 6 人分成 3 个无序小组，每组 2 人，有多少种分法？

答案：

```text
6!/(2!2!2!3!) = 15
```

**题 2：** 10 名学生分到三个固定小组，人数分别为 3、4、3，有多少种？

答案：

```text
C(10,3)C(7,4)C(3,3) = 4200
```

**题 3：** 求 `x1+x2+x3+x4=10` 的非负整数解个数。

答案：

```text
C(10+4-1,4-1) = C(13,3) = 286
```

**题 4：** 求 `x1+x2+x3+x4=10` 的正整数解个数。

答案：

```text
C(10-1,4-1) = C(9,3) = 84
```

**题 5：** 6 个不同球放入 5 个相同盒子，每盒非空，有多少种？

答案：

只能分组类型为：

```text
2+1+1+1+1
```

选哪两个球在同一盒：

```text
C(6,2) = 15
```

## 4.4 容斥原理与错排

### 必须掌握

容斥公式：

```text
|A∪B| = |A| + |B| - |A∩B|

|A∪B∪C|
= |A|+|B|+|C|
- |A∩B|-|A∩C|-|B∩C|
+ |A∩B∩C|
```

关键词：

```text
至少一个
都不
不能在某位置
没有一个满足
```

错排：

```text
D_n = n!(1 - 1/1! + 1/2! - 1/3! + ... + (-1)^n/n!)
```

常见值：

```text
D1=0
D2=1
D3=2
D4=9
D5=44
D6=265
```

### 解题技巧

1. “至少一个满足”常用 `总数 - 一个都不满足`。
2. “所有都拿错”就是错排。
3. “不能出现连续子串”可用总数减相邻块。

### 练习题与答案

**题 1：** 5 个元素全排列中不出现连续子串 `ab`，有多少种？

答案：

```text
5! - 4! = 120 - 24 = 96
```

**题 2：** 6 个元素全排列中不出现连续子串 `ab` 和 `cd`，有多少种？

答案：

```text
6! - 5! - 5! + 4!
= 720 - 120 - 120 + 24
= 504
```

**题 3：** 4 个人各有一顶帽子，随机拿帽子，所有人都拿错有多少种？

答案：

```text
D4 = 9
```

**题 4：** 5 个人随机拿帽子，至少有一个人拿对的情况数是多少？

答案：

```text
5! - D5 = 120 - 44 = 76
```

## 5.1 二项式定理与组合数求和

### 必须掌握

二项式定理：

```text
(a+b)^n = Σ C(n,k)a^(n-k)b^k
```

常用求和：

```text
C(n,0)+C(n,1)+...+C(n,n)=2^n
C(n,1)+C(n,2)+...+C(n,n)=2^n-1
C(n,0)+C(n,2)+C(n,4)+...=2^(n-1)
C(n,1)+C(n,3)+C(n,5)+...=2^(n-1)
```

### 练习题与答案

```text
2^10 - 1 = 1023
```

**题 2：** 求 `(2x+3y)^5` 中 `x^3y^2` 的系数。

答案：

```text
C(5,2) × 2^3 × 3^2 = 10×8×9 = 720
```

**题 3：** 求 `(x-2)^6` 中 `x^4` 项系数。

答案：

```text
C(6,2)(-2)^2 = 15×4 = 60
```

## 5.2 多项式定理

### 必须掌握

对于：

```text
(x1+x2+...+xr)^n
```

项：

```text
x1^a1 x2^a2 ... xr^ar
```

的组合系数为：

```text
n!/(a1!a2!...ar!)
```

其中：

```text
a1+a2+...+ar=n
```

若每一项自带系数，还要把系数的对应幂次乘进去。

### 练习题与答案

**题 1：** 求 `(x+y+z)^6` 中 `x^2y^3z` 的系数。

答案：

```text
6!/(2!3!1!) = 60
```

**题 2：** 求 `(2x+3y+z)^4` 中 `x^2yz` 项系数。

答案：

```text
组合系数：4!/(2!1!1!) = 12
原系数：2^2 × 3 × 1 = 12
总系数：12 × 12 = 144
```

**题 3：** 求 `(3x-y+2z)^3` 中 `xyz` 项系数。

答案：

```text
组合系数：3!/(1!1!1!) = 6
原系数：3 × (-1) × 2 = -6
总系数：6 × (-6) = -36
```

## 5.3 普通母函数与广义二项式

### 必须掌握

基础公式：

```text
1/(1-x) = 1 + x + x^2 + x^3 + ...
```

广义二项式：

```text
1/(1-x)^r = Σ C(r+k-1,k)x^k
```

变形：

```text
1/(1-ax)^r = Σ C(r+k-1,k)a^k x^k
```

所以 `x^k` 的系数：

```text
C(r+k-1,k)a^k
```

### 解题技巧

1. 先把分母写成 `(1-ax)^r`。
2. 确定 `r`、`a`、`k`。
3. 直接套 `C(r+k-1,k)a^k`。

### 练习题与答案

**题 1：** 求 `1/(1-2x)^2` 中 `x^3` 项系数。

答案：

```text
C(2+3-1,3) × 2^3 = C(4,3)×8 = 32
```

**题 2：** 求 `1/(1-3x)^3` 中 `x^2` 项系数。

答案：

```text
C(3+2-1,2) × 3^2 = C(4,2)×9 = 54
```

**题 3：** 求 `1/(1-x)^4` 中 `x^5` 项系数。

答案：

```text
C(4+5-1,5) = C(8,5) = 56
```

**题 4：** 设 `G(x)=(1+x+x^2)/(1-x)`，求 `a0,a1,a2,a3`。

答案：

```text
G(x) = 1 + 2x + 3x^2 + 3x^3 + ...
a0=1, a1=2, a2=3, a3=3
```

## 5.4 指数型母函数 EGF

### 必须掌握

适用场景：

```text
有顺序排列
元素可重复
不同类型元素出现次数有限制
```

基本模型：

```text
出现次数不限制：e^x
至少出现一次：e^x - 1
出现偶数次：(e^x + e^(-x))/2
出现奇数次：(e^x - e^(-x))/2
```

### 解题技巧

1. 每种元素对应一个 EGF 因子。
2. 条件独立时，把各因子相乘。
3. 若需要长度为 n 的排列个数，通常取 `x^n/n!` 的系数再乘 `n!`。

### 练习题与答案

**题 1：** 由数字 1、2、3、4 组成长度为 n 的序列，要求 1 出现偶数次，2 出现奇数次，3 不限制，4 至少出现一次。写出指数型母函数。

答案：

```text
((e^x + e^(-x))/2)
× ((e^x - e^(-x))/2)
× e^x
× (e^x - 1)
```

**题 2：** 用数字 1、2、3 构成 n 位序列，要求数字 2 出现偶数次，1、3 出现次数不限，求个数。

答案：

```text
G(x) = e^(2x) × (e^x + e^(-x))/2
     = (e^(3x) + e^x)/2

个数 = (3^n + 1)/2
```

**题 3：** A、B、C 三类元素组成长度为 n 的序列，要求 A 出现偶数次，B 至少出现一次，C 不限制。写出 EGF。

答案：

```text
((e^x + e^(-x))/2) × (e^x - 1) × e^x
```

## 5.5 一阶线性递推

### 必须掌握

常见形式：

```text
a_n = p·a_(n-1) + q
```

步骤：

```text
1. 解齐次递推 a_n = p·a_(n-1)
2. 找常数特解
3. 写通解
4. 代入初值
```

### 练习题与答案

**题 1：** 已知 `a_n = 3a_(n-1)+2, a_0=1`，求 `a_3`。

答案：

```text
a1 = 3×1 + 2 = 5
a2 = 3×5 + 2 = 17
a3 = 3×17 + 2 = 53
```

**题 2：** 已知 `a_n = 2a_(n-1)-5, a_0=3`，求通项。

答案：

常数特解设为 `A`：

```text
A = 2A - 5
A = 5
```

通解：

```text
a_n = C·2^n + 5
```

代入 `a_0=3`：

```text
C + 5 = 3
C = -2
```

所以：

```text
a_n = 5 - 2^(n+1)
```

## 6.2 树、二叉树、生成树

### 必须掌握

树：

```text
n 个顶点的树有 n-1 条边。
树的度数和 = 2(n-1)。
```

树的叶子公式：

```text
n1 = 2 + Σ(k-2)n_k
```

其中 `n1` 是 1 度顶点数，`n_k` 是 k 度顶点数。

二叉树：

```text
n0 = n2 + 1
```

其中 `n0` 是叶子节点数，`n2` 是度为 2 的节点数。

生成树：

```text
包含所有顶点、连通、无回路、边数为 v-1。
连通图有 v 个顶点、e 条边，得到生成树需删除 e-v+1 条边。
完全图 K_n 的生成树数：n^(n-2)。
```

### 练习题与答案

**题 1：** 一棵树有 12 个顶点，求边数。

答案：

```text
12 - 1 = 11
```

**题 2：** 一棵树有 3 个 2 度顶点、2 个 3 度顶点、1 个 4 度顶点，求叶子数。

答案：

```text
n1 = 2 + (3-2)×2 + (4-2)×1
   = 2 + 2 + 2
   = 6
```

**题 3：** 一棵二叉树有 10 个叶子节点，求度为 2 的节点个数。

答案：

```text
n2 = n0 - 1 = 9
```

**题 4：** 一个连通图有 5 个顶点、7 条边，要得到一棵生成树，需要删除几条边？

答案：

```text
e-v+1 = 7-5+1 = 3
```

**题 5：** 完全图 `K4` 有多少棵生成树？

答案：

```text
4^(4-2) = 16
```

## 6.3 平面图、二部图、着色、最长路

### 必须掌握

平面图欧拉公式：

```text
v - e + f = 2
```

面度数：

```text
所有面的度数之和 = 2e
若每个面的度数至少为 k，则 kf ≤ 2e。
```

若简单连通平面图每个面都是三角形：

```text
3f = 2e
e = 3v - 6
```

二部图：

```text
G 是二部图 ⇔ G 无奇圈
连通二部图且顶点数大于 1，点着色数为 2。
```

Petersen 图：

```text
边着色数 χ'(G)=4。
```

最长路法：

```text
取一条最长路，端点的所有邻点都在这条路上。
若最小度 δ(G)≥k，则图中存在长度至少为 k 的路。
```

### 练习题与答案

**题 1：** 一个连通平面图有 8 个顶点、12 条边，求面数。

答案：

```text
v-e+f=2
8-12+f=2
f=6
```

**题 2：** 一个简单连通平面图有 `n≥3` 个顶点，每个面的度数都是 3，求边数。

答案：

```text
v-e+f=2
3f=2e
联立得 e=3n-6
```

**题 3：** 连通平面图有 12 条边，每个面的度数都不少于 4，问面数最多是多少？

答案：

```text
4f ≤ 2e
f ≤ e/2 = 6
```

**题 4：** 一个简单连通图没有奇圈，且顶点数大于 1，求着色数。

答案：

```text
无奇圈 ⇔ 二部图
连通且顶点数大于 1，所以 χ(G)=2
```

**题 5：** 三角形图 `C3` 是否为二部图？着色数是多少？

答案：

```text
C3 是奇圈，不是二部图。
着色数为 3。
```

# 七、证明题模板

## 7.1 集合相等证明

### 标准写法

证明 `A=B`：

```text
第一步：证明 A⊆B
第二步：证明 B⊆A
结论：A=B
```

也可用元素等价变形：

```text
x∈左边
⇔ ...
⇔ x∈右边
```

### 练习题与答案

**题 1：** 证明 `A-B = A∩B^c`。

答案：

```text
x ∈ A-B
⇔ x∈A 且 x∉B
⇔ x∈A 且 x∈B^c
⇔ x∈A∩B^c
```

所以：

```text
A-B = A∩B^c
```

**题 2：** 证明 `A-(B∪C) = (A-B)∩(A-C)`。

答案：

```text
x ∈ A-(B∪C)
⇔ x∈A 且 x∉(B∪C)
⇔ x∈A 且 x∉B 且 x∉C
⇔ (x∈A 且 x∉B) 且 (x∈A 且 x∉C)
⇔ x∈(A-B)∩(A-C)
```

所以等式成立。

**题 3：** 证明 `(A-B)×C = (A×C)-(B×C)`。

答案：

```text
(x,y) ∈ (A-B)×C
⇔ x∈A-B 且 y∈C
⇔ x∈A 且 x∉B 且 y∈C
⇔ (x,y)∈A×C 且 (x,y)∉B×C
⇔ (x,y)∈(A×C)-(B×C)
```

所以等式成立。

## 7.2 关系性质证明

### 标准写法

证明等价关系：

```text
1. 自反性
2. 对称性
3. 传递性
4. 所以 R 是等价关系
```

证明偏序关系：

```text
1. 自反性
2. 反对称性
3. 传递性
4. 所以 R 是偏序关系
```

### 练习题与答案

**题 1：** 证明恒等关系 `I_A={(x,x)|x∈A}` 既是等价关系，也是偏序关系。

答案：

```text
自反：任意 x∈A，有 (x,x)∈I_A。
对称：若 (x,y)∈I_A，则 x=y，因此 (y,x)=(x,x)∈I_A。
传递：若 (x,y),(y,z)∈I_A，则 x=y 且 y=z，所以 x=z，故 (x,z)∈I_A。
```

所以它是等价关系。

又因为：

```text
若 (x,y)∈I_A 且 (y,x)∈I_A，则 x=y。
```

满足反对称性。因此它也是偏序关系。

**题 2：** 证明整数集上关系 `aRb ⇔ a-b` 能被 3 整除是等价关系。

答案：

```text
自反：a-a=0，能被 3 整除。
对称：若 a-b 能被 3 整除，则 b-a=-(a-b) 也能被 3 整除。
传递：若 a-b 与 b-c 都能被 3 整除，则 a-c=(a-b)+(b-c) 也能被 3 整除。
```

所以 R 是等价关系。

## 7.3 组合恒等式证明

### 必须掌握

常见恒等式：

```text
C(n,r) = C(n,n-r)
Σ_{k=0}^n C(n,k) = 2^n
C(2n,n) = Σ_{k=0}^n C(n,k)^2
```

### 解题技巧

1. 优先用组合意义证明。
2. 左右两边解释为“同一件事的两种计数方法”。
3. 求和式通常对应“分类计数”。

### 练习题与答案

**题 1：** 证明 `C(n,r)=C(n,n-r)`。

答案：

从 n 个元素中选 r 个，等价于决定剩下哪 n-r 个不选，所以：

```text
C(n,r)=C(n,n-r)
```

**题 2：** 证明 `Σ_{k=0}^n C(n,k)=2^n`。

答案：

左边表示从 n 个元素中选任意多个元素：

```text
C(n,0)+C(n,1)+...+C(n,n)
```

右边表示每个元素都有“选/不选”两种状态：

```text
2^n
```

所以恒等式成立。

**题 3：** 证明 `C(2n,n)=Σ_{k=0}^n C(n,k)^2`。

答案：

把 2n 个元素分成两组，每组 n 个。

从 2n 个元素中选 n 个，总数是：

```text
C(2n,n)
```

也可以按第一组取 k 个分类：

```text
Σ_{k=0}^n C(n,k)C(n,n-k)
```

又因为：

```text
C(n,n-k)=C(n,k)
```

所以：

```text
C(2n,n)=Σ_{k=0}^n C(n,k)^2
```

## 7.4 鸽巢原理证明

### 必须掌握

基本形式：

```text
把 n+1 个对象放入 n 个盒子，至少有一个盒子中有不少于 2 个对象。
```

推广形式：

```text
把 N 个对象放入 k 个盒子，至少有一个盒子中有不少于 ceil(N/k) 个对象。
```

解题关键：

```text
对象 = 鸽子
分类 = 鸽巢
```

### 练习题与答案

**题 1：** 证明任意 13 个人中，至少有两个人出生月份相同。

答案：

```text
13 个人是鸽子，12 个月份是鸽巢。
13 个对象放入 12 个盒子，必有一个盒子至少有 2 个对象。
```

所以至少两个人出生月份相同。

**题 2：** 用红、蓝两色给一个 `10×3` 棋盘涂色，证明至少有两行涂色完全相同。

答案：

每行 3 个格子，每格 2 种颜色，一行共有：

```text
2^3 = 8
```

种涂色方式。

10 行放入 8 种涂色方式中，必有两行涂色完全相同。

**题 3：** 从 1 到 20 中任取 11 个整数，证明其中必有两个数的和为 21。

答案：

把 1 到 20 分成 10 组：

```text
{1,20}, {2,19}, {3,18}, ..., {10,11}
```

每组两数之和都是 21。任取 11 个数放入 10 组，必有一组两个数都被选中，所以它们的和为 21。

# 八、综合训练题

## 综合题 1：排列组合

从字母集合：

```text
{a,a,b,b,c,c,d}
```

中选出 4 个字母排列，要求 `c` 至少出现 1 次，求不同排列数。

答案：

按 `c` 出现次数分类。

### 情况一：c 出现 1 次

从 `{a,a,b,b,d}` 中再选 3 个：

```text
aab, aad, abb, abd, bbd
```

对应排列数：

```text
aabc：4!/2! = 12
aacd：4!/2! = 12
abbc：4!/2! = 12
abcd：4! = 24
bbcd：4!/2! = 12
```

小计：

```text
72
```

### 情况二：c 出现 2 次

从 `{a,a,b,b,d}` 中再选 2 个：

```text
aa, ab, ad, bb, bd
```

对应排列数：

```text
aacc：4!/(2!2!) = 6
abcc：4!/2! = 12
adcc：4!/2! = 12
bbcc：4!/(2!2!) = 6
bdcc：4!/2! = 12
```

小计：

```text
48
```

总数：

```text
72 + 48 = 120
```

## 综合题 2：图论

连通平面图 G 有 9 个顶点，顶点度数分别为：

```text
2,2,3,3,3,4,4,5,6
```

求边数和面数。

答案：

```text
总度数 = 2+2+3+3+3+4+4+5+6 = 32
边数 e = 32/2 = 16
```

由欧拉公式：

```text
v-e+f=2

f=9
```

所以：

```text
边数 16，面数 9。
```

## 综合题 3：偏序关系

设：

```text
A = {1,2,3,4,5,6,8,10}
```

关系 R 为整除关系。求：

```text
1. R 中有多少个有序对
2. 写出一条最长链
3. 写出一个反链
```

答案：

逐个统计能整除的元素：

```text
1：1,2,3,4,5,6,8,10，共 8 个
2：2,4,6,8,10，共 5 个
3：3,6，共 2 个
4：4,8，共 2 个
5：5,10，共 2 个
6：6，共 1 个
8：8，共 1 个
10：10，共 1 个
```

所以有序对总数：

```text
8+5+2+2+2+1+1+1 = 22
```

最长链示例：

```text
1 | 2 | 4 | 8
```

反链示例：

```text
{6,8,10}
```
